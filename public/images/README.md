# 이미지 파일 가이드

이 폴더에 실제 이미지 파일을 추가해주세요.

## 📁 필요한 이미지 파일

### 제품 이미지 (products/)
다음 4개 이미지를 `products/` 폴더에 추가:

1. **vegetables.jpg** - 신선 채소 이미지
   - 추천: 상추, 배추, 당근 등 다양한 채소
   - 크기: 최소 800x800px

2. **fish.jpg** - 신선 생선 이미지
   - 추천: 신선한 생선 또는 해산물
   - 크기: 최소 800x800px

3. **fruits.jpg** - 프리미엄 과일 이미지
   - 추천: 오렌지, 사과, 딸기 등
   - 크기: 최소 800x800px

4. **meat.jpg** - 최상급 육류 이미지
   - 추천: 소고기, 돼지고기 등
   - 크기: 최소 800x800px

### 배지 이미지 (badges/)
다음 4개 이미지를 `badges/` 폴더에 추가:

1. **delivery.jpg** - 산지 직송 이미지
   - 추천: 트럭 배송, 농장에서 출발하는 모습

2. **freshness.jpg** - 신선도 보장 이미지
   - 추천: 신선한 식재료, 얼음 박스 등

3. **pricing.jpg** - 가격 투명 이미지
   - 추천: 가격표, 계산기, 돈 이미지

4. **quality.jpg** - 품질 인증 이미지
   - 추천: 인증 마크, 품질 검사 장면

## 🎨 이미지 요구사항

- **형식**: JPG, PNG, WebP
- **권장 크기**:
  - 제품 이미지: 800x800px ~ 1200x1200px (정사각형)
  - 배지 이미지: 800x600px ~ 1200x900px (4:3 비율)
- **용량**: 각 이미지 500KB 이하 (최적화 권장)
- **품질**: 고해상도, 밝고 선명한 이미지

## 📝 이미지가 없을 때

이미지 파일이 없으면 자동으로 **그라데이션 배경**이 표시됩니다.
- 신선 채소: 초록색 그라데이션
- 신선 생선: 파란색 그라데이션
- 프리미엄 과일: 주황색 그라데이션
- 최상급 육류: 빨간색 그라데이션

## 🔄 이미지 추가 방법

1. 이미지를 해당 폴더에 복사
2. 파일명이 정확한지 확인
3. 브라우저 새로고침

예시:
```
public/
  images/
    products/
      vegetables.jpg  ✅
      fish.jpg        ✅
      fruits.jpg      ✅
      meat.jpg        ✅
    badges/
      delivery.jpg    ✅
      freshness.jpg   ✅
      pricing.jpg     ✅
      quality.jpg     ✅
```
