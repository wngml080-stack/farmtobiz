## 📐 와이어프레임 구조 (개발팀 전달용)

| 화면 | 핵심 요소 | 레이아웃 | 비고 |
|------|----------|---------|------|
| **Hero** | 메인 카피, CTA 버튼, 이미지, 신뢰 배지 | 1-column (mobile), Center-aligned | Full-height, 그라데이션 배경 |
| **가격 비교** | 막대 그래프, 가격 데이터, 절감액 강조 | 1-column, Chart centered | 모바일: 테이블, 데스크탑: 차트 |
| **상세페이지** | 상품 이미지, 품질 정보 카드 (3개) | 2-column grid (desktop), 1-column (mobile) | 카드 호버 효과 |
| **편의성** | 스마트폰 목업, 3단계 프로세스 | 2-column grid (desktop), 1-column (mobile) | Navy 배경, 흰색 텍스트 |
| **후기** | 후기 카드 (3개), 통계 (4개), 신뢰 배지 | 3-column grid → 1-column (mobile) | 카드 그림자 효과 |
| **폼** | 입력 필드 (4개), 체크박스, CTA 버튼 | 1-column, Max-width 600px | Green 배경, 흰색 폼 박스 |
| **Footer** | 회사 정보, 팀원, 저작권 | 2-column grid → 1-column (mobile) | Navy 배경, 흰색 텍스트 |
